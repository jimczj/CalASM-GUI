# 异动分析计算器 (CalASM GUI)

这是一个基于 Python 的 A 股严重异动计算与预测工具，提供图形化界面（GUI），方便在 Windows 平台上进行批量分析。

## 📅 功能特性

*   **异动分析**：基于历史数据和实时行情，计算股票的严重异动偏离值。
*   **T+X 预测**：推算未来三天（T+1, T+2, T+3）的理论触线价格、剩余空间及允许连板数。
*   **批量处理**：支持一次性输入多只股票代码进行批量分析。
*   **实时数据**：如当日数据未更新，会自动抓取盘中实时数据进行补全。
*   **图表生成**：自动生成分析结果图片及总览表图片，保存在 `images/` 目录下。
*   **便携运行**：支持打包为独立的 `.exe` 文件，无需 Python 环境即可在 Windows 上运行。

## 🛠️ 安装与运行

### 方式一：直接运行 Python 脚本

1.  **环境准备**：
    确保已安装 Python 3.8 或以上版本。

2.  **安装依赖**：
    在项目根目录下打开终端，运行以下命令安装所需库：
    ```bash
    pip install -r requirements.txt
    ```

3.  **运行程序**：
    ```bash
    python calASM_gui.py
    ```

## 📖 使用说明

1.  **输入股票**：在界面上方的文本框中输入股票列表。
    *   格式：`股票代码 股票名称` （每行一个）
    *   例如：
        ```text
        600372 中航机载
        601698 中国卫通
        ```
2.  **开始分析**：点击“开始分析”按钮。
3.  **结果查看**：
    *   **日志窗口**：界面下方会实时显示分析进度和简单的文字版汇总表。
    *   **图片结果**：勾选“同时保存图片”后，详细的分析表格图片会保存在程序运行目录下的 `images/` 文件夹中。

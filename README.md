# 异动分析计算器 (CalASM GUI)

这是一个基于 Python 的 A 股分析工具集，包含两个主要工具：**异动分析计算器** 和 **最高价反查工具**。提供图形化界面（GUI），方便在 Windows 平台上进行数据分析。

## 📦 包含工具

1.  **异动分析计算器 (`calASM_gui.py`)**：计算股票严重异动偏离值及未来空间。
2.  **最高价反查工具 (`findStoke_gui.py`)**：根据当日最高价反向查找对应的股票。

---

## 📅 Tool 1: 异动分析计算器

### 功能特性

*   **异动分析**：基于历史数据和实时行情，自动计算股票的严重异动偏离值（10日100%、30日200%）。
*   **灵活预测**：支持自定义预测天数（T+X），推算未来的理论触线价格、剩余空间。
*   **智能综判**：自动生成综合最严异动列表，取不同规则下的极小值空间。
*   **批量处理**：支持一次性输入多只股票代码进行批量分析。
*   **实时补全**：盘中自动抓取实时数据补全当日K线，确保计算实时性。
*   **图表生成**：自动生成精美的分析结果表格图片及总览图，保存在 `images/` 目录下。

### 使用说明

1.  **运行程序**：
    ```bash
    python calASM_gui.py
    ```
2.  **输入股票**：在界面上方输入股票列表（格式：`代码 名称` 或仅 `代码`，每行一个）。
3.  **参数设置**：
    *   **预测天数**：默认为 3 天，可根据需要调整（建议 10 天以内）。
    *   **显示连板**：勾选后会在结果中显示推算的允许连板数。
    *   **保存图片**：建议勾选，结果更直观。
4.  **查看结果**：点击“开始分析”，结果将显示在下方日志栏，图片保存在 `images/` 中。

---

## 🔍 Tool 2: 最高价反查工具

当您只知道某个股票当天的“最高价”，却不知道具体是哪支股票时，可以使用此工具快速查找。

### 功能特性
*   **全市场搜索**：基于实时行情，扫描所有 A 股。
*   **精确匹配**：查找当日最高价与输入价格一致的股票。

### 使用说明

1.  **运行程序**：
    ```bash
    python findStoke_gui.py
    ```
2.  **输入价格**：在输入框中输入目标最高价（例如 `24.58`）。
3.  **开始查找**：点击“查找股票”或按回车键。
4.  **结果列表**：列表将显示所有最高价匹配的股票代码、名称、现价及涨跌幅。
